# Aplikasi Manajemen Tugas (Task Management Web App)

Aplikasi manajemen tugas berbasis web yang responsif, dirancang untuk membantu pengguna menyusun, melacak, dan mengelola daftar pekerjaan harian mereka agar lebih produktif.

Live Demo Aplikasi Klik Di Sini


# Fitur Utama
* Tambah Tugas: Pengguna dapat memasukkan tugas baru ke dalam daftar.
* Hapus Tugas: Menghapus tugas yang sudah selesai atau tidak diperlukan lagi.
* Penyimpanan Otomatis (Persistensi Data): Menggunakan LocalStorage sehingga data tugas tetap tersimpan dan tidak hilang meskipun halaman web dimuat ulang (refresh) atau browser ditutup.
* Desain Responsif: Tampilan web yang rapi dan nyaman dilihat, baik melalui komputer maupun smartphone.



# Teknologi Yang Digunakan
* HTML5: Membuat struktur dasar, form input, dan area daftar tugas.
* CSS3: Mengatur tata letak modern menggunakan Flexbox, serta memberikan animasi visual yang halus.
* JavaScript: Menangani logika manipulasi DOM (Data Object Model), pengolahan data array, dan integrasi API LocalStorage.



# Apa Yang Saya Pelajari dari Proyek Ini?
Melalui proyek ini, saya berhasil memahami dan menerapkan beberapa konsep dasar penting dalam pengembangan web:
1. Manipulasi DOM: Cara menghubungkan elemen HTML dengan JavaScript untuk membuat halaman web yang interaktif.
2. Penyimpanan Lokal (Browser Storage): Memahami cara kerja penyimpanan data di sisi klien (client-side) menggunakan format data JSON.
3. Manajemen State Sederhana: Mengelola perubahan data pada aplikasi (menambah dan menghapus) secara sinkron antara memori array dan tampilan layar.
